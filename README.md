## 🧪 My AI Engineering Principles

```
1. Eval before you ship.          No LLM feature without a test set and baseline.
2. Cost is a feature.             Track tokens. Set budgets. Log everything.
3. Explainability > magic.        Users trust what they can understand.
4. Multilingual from day one.     Good embeddings make language a solved problem.
5. The notebook is not the app.   FastAPI, Docker, CI/CD — or it didn't ship.
```
